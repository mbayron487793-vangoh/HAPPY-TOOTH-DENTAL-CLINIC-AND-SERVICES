# Poppins Font Installation

For the best visual experience, download the Poppins font family from Google Fonts and place the following files in this directory:

1. **Poppins-Regular.ttf**
2. **Poppins-Medium.ttf**
3. **Poppins-SemiBold.ttf**
4. **Poppins-Bold.ttf**
5. **Poppins-Light.ttf**

## Download Link
https://fonts.google.com/specimen/Poppins

## Instructions
1. Visit the link above
2. Click "Download family"
3. Extract the ZIP file
4. Copy the .ttf files listed above to this `fonts` folder

If fonts are not installed, the application will automatically fall back to Segoe UI (Windows) or a similar system font.
